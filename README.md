# CSS-GRID

A CSS-GRID udemy project made with

```
1 HTML
2 CSS
3.CSS GRID
4 FLEXBOX

```


## Installation via github:

1. Clone the github project.
2. `cd CSS-GRID-master`
3. Run `npm install && npm start`
